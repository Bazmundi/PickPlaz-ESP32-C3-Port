Detailed Description
--------------------

Provides GPIO, timers, PWM, UART, SPI, I2C, ADC, and optional self-test services using ESP-IDF drivers. The HAL hides platform specifics from the application layer.

Thread-safety:

- Not thread-safe; callers should serialize access when sharing resources.
