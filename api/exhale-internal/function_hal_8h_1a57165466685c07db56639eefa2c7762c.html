Function Documentation¶

void hal_delay_ms(uint32_t ms)

Delays execution for the requested number of milliseconds.

Uses the FreeRTOS tick delay to suspend the calling task.

Preconditions:

  • Must be called from a task context (not an ISR).

Side effects:

  • Suspends the current task.

Parameters:

ms – Delay duration in milliseconds. Zero performs no delay.
