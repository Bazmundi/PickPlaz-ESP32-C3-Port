Function hal_gpio_read(int)¶

  • Defined in File hal.h

Function Documentation¶

hal_gpio_level_t hal_gpio_read(int pin)

Reads the current logic level of a GPIO.

Returns HAL_GPIO_LOW if the pin is invalid.

Parameters:

pin – GPIO number to read.

Returns:

HAL_GPIO_HIGH or HAL_GPIO_LOW based on the sampled level.
