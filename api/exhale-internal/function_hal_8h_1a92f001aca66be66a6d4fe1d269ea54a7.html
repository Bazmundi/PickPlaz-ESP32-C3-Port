Function hal_pwm_set_duty(int, uint32_t)¶

Function Documentation¶

hal_status_t hal_pwm_set_duty(int channel, uint32_t duty)

Updates the PWM duty for a configured channel.

Clamps duty to the configured maximum for the channel before updating.

Preconditions:

  • channel must be configured via hal_pwm_init().

Side effects:

  • Updates LEDC duty and commits it to hardware.

Parameters:

  • channel – LEDC channel index.

  • duty – Duty value in channel resolution units.

Returns:

HAL_OK on success, HAL_ERR_INVALID on invalid params or failure, HAL_ERR_UNSUPPORTED if the channel is not configured.
