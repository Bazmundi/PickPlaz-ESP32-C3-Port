int hal_i2c_write(int bus_id, uint8_t addr, const uint8_t *data, size_t length)¶

Writes data to an I2C device.

Uses a blocking transfer with a 100 ms timeout.

Preconditions:

  • I2C bus has been initialized.

Parameters:

  • bus_id – I2C bus identifier (must be I2C_NUM_0).

  • addr – 7-bit I2C device address.

  • data – Buffer to write. Must not be NULL.

  • length – Number of bytes to write. Must be > 0.

Returns:

Number of bytes written, or HAL_ERR_INVALID on failure, HAL_ERR_UNSUPPORTED if the bus is not available.
