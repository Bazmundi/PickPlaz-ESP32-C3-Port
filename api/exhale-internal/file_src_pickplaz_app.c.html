Detailed Description¶

Owns the 1 kHz application tick, translates button/opto/feed inputs into state transitions, and drives LED/motor outputs through the HAL. This module is platform-agnostic and relies on board pin mappings defined in board_pins.h plus optional overrides in hal_config.h.

Thread-safety:

  • Not thread-safe; intended to run from a single periodic tick.
