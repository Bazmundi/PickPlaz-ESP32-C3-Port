void hal_selftest_run(void)

Runs a basic HAL self-test sequence.

Validates GPIO, tick timer, PWM, and optional peripheral bring-up. Only compiled when HAL_SELFTEST is defined.

Side effects:

  • Reconfigures GPIO/PWM peripherals for test patterns.

  • Writes test output to the UART when available.

Note

Intended for development builds only.
