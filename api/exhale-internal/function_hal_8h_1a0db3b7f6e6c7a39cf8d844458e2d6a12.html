Function hal_gpio_write(int, hal_gpio_level_t)¶

Function Documentation¶

hal_status_t hal_gpio_write(int pin, hal_gpio_level_t level)

Writes a logic level to a GPIO output.

Preconditions:

  • pin must be configured as an output-capable GPIO.

Side effects:

  • Updates the GPIO output level.

Parameters:

  • pin – GPIO number to write.

  • level – Logic level to drive.

Returns:

HAL_OK on success, HAL_ERR_INVALID on invalid pin.
