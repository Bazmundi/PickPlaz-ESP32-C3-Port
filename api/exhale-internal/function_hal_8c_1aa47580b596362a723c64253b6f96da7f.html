Function Documentation¶

hal_status_t hal_gpio_config_output(int pin, hal_gpio_level_t initial_level)¶

Configures a GPIO pin as a push-pull output.

Sets the pin to output mode with no pull resistors and applies the initial logic level.

Preconditions:

  • pin must be a valid output-capable GPIO.

Postconditions:

  • GPIO is configured for output and driven to initial_level.

Side effects:

  • Updates GPIO hardware configuration.

Parameters:

  • pin – GPIO number to configure.

  • initial_level – Initial logic level to drive.

Returns:

HAL_OK on success, HAL_ERR_INVALID on invalid pin/config failure.
