Function Documentation¶

static void app_set_motor(uint32_t pwm, bool forward)¶

Drives the motor H-bridge outputs with scaled PWM.

Applies the STM32-style duty value (0..2048) to the ESP32 LEDC channels. Forward drives IN2, backward drives IN1, matching the STM32 polarity model.

Preconditions:

  • Motor PWM channels are initialized via hal_pwm_init().

Side effects:

  • Updates LEDC duty on the motor output channels.

Parameters:

  • pwm – Duty value in STM32 units (0..2048).

  • forward – True for forward direction, false for reverse.
