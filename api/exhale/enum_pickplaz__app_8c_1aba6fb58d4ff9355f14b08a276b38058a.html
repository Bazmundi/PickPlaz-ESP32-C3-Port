enum button_event_t¶

Represents the debounced button event for a single tick.

Values:

enumerator BUTTON_none¶

No event this tick.

enumerator BUTTON_short¶

Short press released after debounce timing.

enumerator BUTTON_long¶

Long press released after exceeding the long-press threshold.

enumerator BUTTON_hold¶

Button remains held; emitted while pressed.
