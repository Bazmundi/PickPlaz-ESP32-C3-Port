Function app_tick¶

  • Defined in File pickplaz_app.c

Function Documentation¶

static void app_tick(void *user_data)¶

Executes one 1 kHz application tick.

Samples buttons, updates opto/feed state, advances the application and motor FSMs, and refreshes LED outputs.

Preconditions:

  • pickplaz_app_init() has configured IO and initialized state.

Side effects:

  • Reads GPIO/ADC inputs and updates PWM/GPIO outputs.

Parameters:

user_data – Unused; reserved for future tick context.
