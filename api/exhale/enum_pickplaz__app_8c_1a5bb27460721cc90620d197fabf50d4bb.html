enum app_state_t¶

Represents the main application state machine.

States mirror the STM32 firmware behavior for indexed moves and continuous motion while a button is held.

Values:

enumerator APP_init¶

Initial state after reset.

enumerator APP_idle¶

Idle state waiting for button or feed events.

enumerator APP_increment_forward1¶

Forward step: move until the opto leaves index or timeout.

enumerator APP_increment_backward1¶

Reverse step: move until the opto leaves index or timeout.

enumerator APP_increment_forward2¶

Forward step: continue until opto returns to index or timeout.

enumerator APP_increment_backward2¶

Reverse step: continue until opto returns to index or timeout.

enumerator APP_free_forward¶

Continuous forward motion while the forward request is held.

enumerator APP_free_backward¶

Continuous reverse motion while the reverse request is held.
