Function hal_tick_start¶

  • Defined in File hal.h

Function Documentation¶

hal_status_t hal_tick_start(uint32_t hz, hal_timer_callback_t callback, void *user_data)

Starts the global HAL tick timer.

Configures a periodic ESP timer that calls the provided callback at the requested frequency.

Preconditions:

  • hz must be > 0.

  • callback must be non-null.

Postconditions:

  • The tick callback is invoked at approximately hz.

Side effects:

  • Allocates and starts an ESP timer.

Parameters:

  • hz – Tick frequency in Hertz.

  • callback – Function to invoke on each tick. Must not be NULL.

  • user_data – Opaque pointer passed to callback.

Returns:

HAL_OK on success, HAL_ERR_INVALID on invalid params or failure.
