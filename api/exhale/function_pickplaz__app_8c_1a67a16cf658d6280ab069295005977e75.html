Function eval_led_feed¶

  • Defined in File pickplaz_app.c

Function Documentation¶

static void eval_led_feed(void)¶

Drives the feed indicator LED.

Emits a 500 ms pulse when a feed event is detected. Uses LED4 when available; otherwise falls back to LED3 to preserve visibility.

Side effects:

  • Updates LED4 GPIO or LED3 PWM output.
