Function app_button_update¶

  • Defined in File pickplaz_app.c

Function Documentation¶

static button_event_t app_button_update(app_button_t *button)¶

Samples and debounces a button, returning edge events.

Implements the STM32 timing model: a debounce counter followed by press timing. Returns BUTTON_short/BUTTON_long on release and BUTTON_hold while the button remains held.

Preconditions:

  • button must be non-null and initialized with pin and polarity.

Postconditions:

  • Debounce and press counters are updated.

Side effects:

  • Reads GPIO through the HAL.

Parameters:

button – Button state storage. Must not be NULL.

Returns:

Button event for this tick.
