Function Documentation¶

hal_status_t hal_i2c_init(int bus_id, uint32_t clock_hz)

Initializes the I2C master bus.

Only I2C_NUM_0 is supported. Pins are defined in hal_config.h.

Preconditions:

  • bus_id must be I2C_NUM_0.

  • SDA/SCL pins must be configured and valid.

Postconditions:

  • I2C driver is installed and ready for transfers.

Side effects:

  • Allocates I2C driver resources.

Parameters:

  • bus_id – I2C bus identifier (must be I2C_NUM_0).

  • clock_hz – Bus clock in Hertz. Zero uses HAL_I2C0_CLOCK_HZ.

Returns:

HAL_OK on success, HAL_ERR_UNSUPPORTED on unsupported config, HAL_ERR_INVALID on invalid parameters or driver failure.
